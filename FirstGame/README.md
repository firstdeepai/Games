# RocketOrbit - My First Game

Welcome to **RocketOrbit**! Yeh meri pehli game hai jo maine Unity seekhte hue banayi hai. Is game mein aap ek rocket ko control karte hain aur obstacles se bachate hue aage badhte hain.

## Controls

Game khelne ke liye ye keys use karein:

| Key | Action |
|-----|--------|
| **A / D** | Left aur Right move karne ke liye |
| **Space** | Rocket ko thrust/fly dene ke liye |

---

## Cheat Codes (Debugging ke liye)

| Code | Function |
|------|----------|
| **C** | Collisions On/Off karne ke liye |
| **N** | Next level par switch karne ke liye |

## HOW TO PLAY?

1. **Download** karein aur game folder ko **extract** karein
2. **RocketOrbit.exe** ya **Polished-Version/RocketOrbit.exe** par double-click karke run karein
3. **Maze karein!** 🎉
